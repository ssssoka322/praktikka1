мой сайт называется people's lives
это новостной сайт, там можно: найти новости, сохранить их, потом посмотреть свои сохраненные новости.
чтобы скачать и открыть проектик, нужно:
1.открыть мой акк на GitHub.
2.нажать зеленую кнопку «Code».
3.скопировать HTTPS-ссылку.
4.открыть терминал на компьютере.
5.ввести команду `git clone скопированная_ссылка.git`.
6.нажать Enter — проект скачается.
7.открыть папку с проектом.
8.дважды кликнуть по файлу index.html, чтобы запустить сайт в браузере.
либо можно скачать ZIP проекта, зайти в VScode. здесь мы должны открыть терминал, написать в него npm start. теперь либо зажать ctrl и нажать на ссылку, либо скопировать ее и вставить в браузер.
